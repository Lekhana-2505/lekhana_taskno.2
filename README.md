# Tribute Page 🌟

This project is a simple Tribute Page built using HTML and CSS. It is designed to honor a person I admire by showcasing their achievements, qualities, and inspiring life story in a visually appealing format.

## 📌 Project Overview

The tribute page presents information about a chosen personality with proper structure, styling, and images. It focuses on creating a clean and attractive layout using basic HTML and CSS skills.

## ✨ Features

- Simple and clean design  
- Well-structured layout using HTML  
- Image section to highlight the person  
- Proper use of headings and paragraphs  
- Basic styling with CSS for better appearance  

## 🛠️ Technologies Used

- HTML5  
- CSS3
